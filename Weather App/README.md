# Weather App (CLI) using Python

## Project Description

This is a simple Command Line Interface (CLI) Weather Application developed using Python. The application fetches real-time weather information for a city using the OpenWeatherMap API.

## Features

* Get current weather information
* Display temperature in Celsius
* Display humidity
* Display weather condition
* Easy to use command-line interface

## Technologies Used

* Python
* Requests Library
* OpenWeatherMap API

## Project Structure

WeatherApp/

├── weather.py

└── README.md

## Requirements

Install the required package:

pip install requests

## API Key Setup

1. Create an account on OpenWeatherMap.
2. Generate an API Key.
3. Replace the API key in the code:

API_KEY = "YOUR_API_KEY"

## How to Run

Open terminal and execute:

python weather.py
